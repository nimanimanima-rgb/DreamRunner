# Dream Runner

Dream Runner is a browser-based third-person movement game inspired by the peaceful endless driving experience of Slow Roads.

Instead of driving, the player runs, jumps, glides, and explores an endless dreamlike natural world.

## Core Fantasy

Run impossibly fast. Jump impossibly high. Move through a calm, endless landscape with no pressure, no enemies, and no hard failure state.

## Target Platform

Browser first.

## Engine

Godot 4.

## Current Goal

Build a small playable prototype where the movement feels good before adding procedural terrain.

## Current Build Status

The first playable movement prototype has been created. The current build includes:

- Godot 4 project
- main scene
- flat ground
- basic player controller
- WASD movement
- sprint
- high jump
- mouse camera

## Next Milestone

**Movement Laboratory:** tune running, jump, air control, camera, and debug UI.
