# Dream Runner — Game Design Document

## One-Sentence Pitch

A browser-based third-person dream-running experience where the player runs fast, jumps high, glides through the air, and explores an endless natural world.

## Inspiration

- Slow Roads
- dreamlike running/flying dreams
- peaceful traversal games
- atmospheric exploration games

## Core Feeling

The game should feel:

- fast
- free
- floaty
- calm
- lonely
- beautiful
- dreamlike

## Core Loop

Run → jump → glide → land → discover new terrain → keep moving.

## Player Movement

The player should be able to:

- run at high speed
- jump much higher than realistic human limits
- control movement while airborne
- land smoothly
- possibly glide or float later

## Camera

Third-person camera.

The camera should:

- follow smoothly
- slightly pull back at high speed
- feel cinematic but controllable
- avoid shaking or nausea
- make jumping feel large and freeing

## World

The world should feel natural and endless.

Core elements:

- rolling hills
- grass
- trees
- boulders
- obstacles
- open areas
- valleys
- soft fog
- sky atmosphere

## Gameplay Rules

The game is not about punishment.

Avoid:

- enemies
- combat
- death loops
- complex scoring
- stressful timers

Focus on:

- movement feel
- atmosphere
- exploration
- freedom

## MVP

The first playable version should include:

- one controllable third-person character
- high-speed running
- high jumping
- smooth camera
- basic terrain
- simple trees/rocks as placeholders
- browser export test

## Current Build Status

The first playable movement prototype has been created. The current build includes:

- Godot 4 project
- main scene
- flat ground
- basic player controller
- WASD movement
- sprint
- high jump
- mouse camera

## Next Milestone

**Movement Laboratory:** tune running, jump, air control, camera, and debug UI.

## Later Features

Possible future features:

- infinite procedural terrain
- gliding
- day/night cycle
- weather
- multiple biomes
- music system
- collectible memories or dream fragments
- photo mode
